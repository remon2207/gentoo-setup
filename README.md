# gentoo-setup

#### USAGE:
`./gentoo-setup/install.sh` or

`./gentoo-setup/install.sh --help`

```bash
./gentoo-setup/install.sh --disk <disk> \
--microcode <microcode> \
--gpu <gpu> \
--user-password <user-password> \
--root-password <root-password>

# example
./gentoo-setup/install.sh --disk /dev/sda \
--microcode intel \
--gpu nvidia \
--user-password 12345 \
--root-password password
```
